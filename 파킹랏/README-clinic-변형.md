# 라엘 피부과의원 — 디자인 5변형 (clinic-v1 ~ v5)

피부과 데모(포트폴리오 카드 #09 "라엘 피부과의원")의 **디자인 변형 세트**.
CLAUDE.md §6 "업종당 5 변형" 규칙의 산출물이며, **5개 모두 같은 피부과**의
다른 톤·스크롤 강도 버전이다. 버린 파일이 아니라 **고객 선택용 변형**이다.

> 의료/피부과 업종은 스크롤을 절제하는 게 신뢰에 유리(CLAUDE.md 5-1b).
> 그래서 변형 간 차이는 화려함보다 **톤앤매너 + 모션 강약**으로 둔다.

## 변형 목록

| 파일 | 변형명 | 톤앤매너 | 스크롤 강도 | 상태 |
|---|---|---|---|---|
| `clinic-v1-clean.html` | Clean | 화이트·미니멀, 정보 우선 | 하향(절제) | 보관(미노출) |
| `clinic-v2-luxury.html` | Luxury | 프리미엄·럭셔리 | 표준 | **대표 — 포트폴리오 노출** |
| `clinic-v3-soft.html` | Soft | 부드러운·따뜻한 | 표준 | 보관(미노출) |
| `clinic-v4-trendy.html` | Trendy | 트렌디·세련 | 상향(적극) | 보관(미노출) |
| `clinic-v5-authority.html` | Authority | 권위·신뢰 강조 | 표준 | 보관(미노출) |

## 노출 규칙

- 포트폴리오 허브([../js/app.js](../js/app.js))에는 **대표안 `clinic-v2-luxury` 1개만** 링크한다.
- 나머지 4개는 **상담 시 고객에게 직접 보여주는 변형 후보**다(셀링포인트: "디자인 5변형 제공").
- 대표안을 바꾸려면 app.js 카드 #09의 `href`만 교체하면 된다.

## 정리/삭제 판단

- 4개 미노출 변형은 **삭제 대상이 아니다.** "5변형 제공"이라는 상품 약속의 실물 근거다.
- 단, 영업에서 실제로 쓰지 않는 변형이 생기면 이 표의 "상태"를 업데이트한다.
