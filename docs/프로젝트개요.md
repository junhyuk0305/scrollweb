# 프로젝트 개요 — 스크롤 웹 템플릿 외주 사업

> 이 프로젝트를 처음 보는 사람(또는 미래의 나)이 **한 번에 이해하기 위한 단일 문서**.
> 작업 규칙은 [CLAUDE.md](../CLAUDE.md), 세부 기획은 각 자산 파일 참고.

---

## 한 줄 정의

**스크롤트리거 기반 프리미엄 인터랙티브 웹사이트를, 직접 만든 핵심 데모 5종(+ 업종별 변형)
라인업을 무기로 빠르게 제작·판매하는 1인 웹 외주 사업.**

---

## 무엇을 파는가

업종별로 최적화된 **고퀄 인터랙티브 랜딩/기업 사이트**를 외주로 제작해 판매한다.
단순 템플릿 갈아끼우기가 아니라, *예시 데모를 보여주고 → 고객 업종에 맞춰
발전·맞춤화*하는 방식이다.

- BASIC(원페이지): **3일** / STANDARD(멀티·시그니처 애니): **5일** / PRO(풀 인터랙션·기능): **7일**

## 3대 차별점

| # | 강점 | 의미 |
|---|---|---|
| 1 | **빠른 납기** | 공통 스택 재사용으로 3일~1주. 시장 평균보다 빠름 |
| 2 | **스크롤트리거 인터랙션** | 애플·토스급 움직임. 저가 외주(템플릿 복제)와 명확히 차별화되는 프리미엄 가치 |
| 3 | **예시 기반 최소 보장** | 핵심 데모로 퀄리티 하한선을 먼저 증명 → 고객 불안 해소, 단가 방어 |

## 기술 토대

- **Lenis**(부드러운 스크롤) + **GSAP/ScrollTrigger**(트리거) + **SplitType**(텍스트 분리)
- 반응형 · `prefers-reduced-motion` 접근성 폴백 · SEO 기본기 · 고정 CTA를 기본 탑재
- 상세 셋업·효과별 코드는 [00_scroll.md](../_catalog/00_scroll.md)

## 사업 모델 (판매 채널)

1. **1단계 — 크몽(상품형)**: "스크롤 애니메이션 랜딩페이지 3일" 같은 상품 등록,
   후기·포트폴리오 축적. 수수료 약 13.2%.
2. **2단계 — 위시켓(프로젝트형)**: 포트폴리오를 무기로 중대형 프로젝트 입찰, 단가 상승.
   수수료 20~25%.
- 견적은 **실수령 역산**(받을 돈 ÷ 0.87 등)으로 책정, **VAT 별도** 표기.
- 계약·배포·인수인계·세무는 별도 비즈니스 지식으로 관리(코드와 분리).
