## 선정 기준 (왜 이 5종인가)

리서치 + 상품 제약(**가격 ~100만원 이하 · 개발 7일 이하 · 1인 제작**)을 모두 통과한 산업만 선정:

1. **전환 확률** — 사이트가 없거나(세그먼트 B) 구식인(세그먼트 A) 업종 우선
2. **제작 난이도** — 무거운 커스텀 백엔드 없이 외부 예약 임베드/폼 UI로 7일 내 완성 가능
3. **내 강점 활용** — 스크롤 인터랙션이 가치가 되는 업종은 적극, 신뢰가 우선인 업종은 절제
4. **포트폴리오 범위** — 5종이 스크롤 강도·디자인 톤을 전부 다르게 → "어떤 업종이든 맞춘다" 증명

## 공통 빌드 규칙 (5종 전부 적용)

- **멀티페이지 + 데모당 폴더 구조**: 각 데모(변형)는 자체 폴더에 페이지별 `*.html`
  (홈·소개·서비스·정보·갤러리·문의 등 업종별 분할) + `assets/styles.css` + `assets/app.js`로
  **html/css/js를 분리**해 구성한다. **원페이지 지양.** GSAP+ScrollTrigger+Lenis CDN, Pretendard 폰트.
- **업종당 5 변형(디자인·스크롤 강도 옵션)**: 고객이 고르도록 업종마다 5개 변형 제작 —
  **3종 = 표준 스크롤 강도(디자인만 다르게) · 1종 = 한 단계 상향 · 1종 = 한 단계 하향.**
  폴더: `<데모명>/v1-슬러그/ … v5-슬러그/`, 각 변형은 독립 배포 가능한 멀티페이지 사이트.
- 외부 이미지는 picsum.photos 고정 ID 또는 CSS 그라디언트 플레이스홀더로 대체(실제 판매 시 교체 안내).
- **모바일 우선 반응형** + **고정 CTA**(전화·카카오·예약/견적 중 업종 맞춤).
- **접근성**: `prefers-reduced-motion` 분기 — 스크럽/패럴랙스 끄고 최종 상태 즉시 노출.
- **성능**: `transform/opacity`만 애니메이트, 60fps 유지, 무거운 효과는 주인공 1~2개만.
- **콘텐츠**: 가상의 현실적인 한국 브랜드/문구로 채움(스톡 느낌 금지). 신뢰 신호 전면 배치.
- 상단에 `<!-- 판매 시 교체: ... -->` 주석으로 교체 포인트(로고·사진·연락처·예약링크) 표기.

## 1. Reverie Stay — 펜션·풀빌라·독채숙소 (스크롤 높음)

- **가상 브랜드**: "고요재(古耀齋)" — 강원도 양양 독채 풀빌라
- **타겟 고객의 진짜 니즈**: 네이버/야놀자 수수료 회피 + 자체 직예약, 공간의 격 전달
- **섹션 구성**:
  1. 풀스크린 히어로(풍경 영상/이미지 + 상호 + "예약하기" 고정 CTA)
  2. 인트로 카피(브랜드 무드, 한 줄 철학) — 단어 단위 페이드업
  3. 객실 소개(2~3 객실 카드, 패럴랙스 이미지, 정원·인원·기준요금)
  4. 공간/어메니티(풀·바베큐·자쿠지 아이콘 리스트 + 갤러리 라이트박스)
  5. 주변 가볼 곳(스팟 3~4, 거리/소요시간)
  6. 예약 안내(달력형 예약 UI 목업 + 네이버예약/전화 버튼, 환불규정)
  7. 오시는 길(지도 임베드 자리 + 주소·주차)
- **스크롤 효과**: 풀스크린 축소(Shrink&Lock) 또는 패럴랙스 풍경, 객실 이미지 진입 리빌,
  배경색 웜톤 전환, 숫자 카운트(누적 투숙·평점)
- **톤**: 자연·여백·고급 웜톤(딥그린/베이지/우드). 세리프 제목 + 넉넉한 여백
- **신뢰 신호**: 실제 후기 평점, 누적 투숙객, 사업자/숙박업 신고번호, 환불규정 명시

## 2. Atelier Build — 인테리어·리모델링·시공 (스크롤 중간)

- **섹션 구성**:
  1. 히어로(대표 시공 풀블리드 + "무료 견적 받기" 고정 CTA)
  2. 브랜드 한 줄 + 핵심 수치(시공 건수·연차·재발주율 카운트업)
  3. **비포/애프터 슬라이더**(드래그 핸들, 2~3 사례) ← 이 업종의 킬러 기능
  4. 시공 포트폴리오 갤러리(공간유형 필터: 아파트/상가/카페/사무실 + 라이트박스)

  6. 예산/공정 안내(평형·범위별 가이드 표)
  7. **견적 문의 폼**(평형·예산·공사범위·연락처) + 카카오 채널
  8. 신뢰 푸터(건설업 등록번호·면허·하자보증·사업자번호)

  포트폴리오 그리드 stagger. (과한 모션 금지 — 신뢰 우선)
- **톤**: 모던 미니멀 + 하이엔드 한 스푼(차콜/오프화이트/우드 액센트)
- **신뢰 신호**: 종합/실내건축 면허, 시공능력평가, 하자보증 기간, 실사 사진
