# 크몽 상품 구성 — 오버스펙 스크롤 홈페이지 (무료 호스팅·고객 계정)

> **단일 기준 문서.** 크몽 인기 전문가 24개+ 기그 리서치(2026-06-05) 기반.
> **노선: 저가 빠른 회전.** 확정 **100만원 이하(30~80만)**. 무기는 **경쟁사가 못 내는 오버스펙 스크롤 애니메이션**.
> 기술: **Plain HTML/CSS/JS + GSAP/Lenis/ScrollTrigger(CDN) + 무료 호스팅(고객 계정, 제공처 검토중) + 문의는 구글시트/이메일 수신(서버·DB 없음)**. 프레임워크·백엔드 없음 = 가볍고·안 깨지고·저관리.
> 핵심 철학: **관리 부담 최소화 = 정적 + 셀프 운영 + 백엔드 최소.** 가격을 낮게 유지하려면 사후 관리가 가벼워야 한다.

## 0. 노선 결정 (왜 저가·오버스펙인가)

- **저가(30~80만)** = 빠른 의사결정·빠른 회전·낮은 CS. 높은 가격은 그만큼 관리 책임이 커져 못 받친다.
- **무기는 가격이 아니라 애니메이션.** 그누보드·아임웹·워드프레스 군은 절대 못 내는 **GSAP 스크롤 인터랙션**으로 "30만원짜리인데 300만원처럼 보이는" 체감을 만든다 → 후기·포트폴리오로 빠른 수익.
- **관리 부담 0 설계**: 회원/로그인/결제/복잡한 관리자 = **안 판다**(CS 지옥). 콘텐츠는 정적 + 고객 셀프 수정 + 건별 유지보수.
- **호스팅비 0원**: 고객 본인 무료 계정에 배포. 우리가 호스팅을 떠안지 않음 = 반복 비용·책임 없음.

## 1. 호스팅 (★검토중 — 제공처 미확정)

> plain HTML 정적이라 어디든 올라간다. 아래 후보 중 1곳을 Phase 0 전까지 확정. 공통 조건: **무료 + 고객 계정 + 빠름**.

| 후보 | 무료 한도 | 장점 | 주의 |
|---|---|---|---|
| **Cloudflare Pages** | 대역폭 무제한 | 글로벌 CDN(빠름)·상업용 OK | 가입 영문 |
| **Vercel (Hobby)** | 100GB/월(≈월 2.5~5만 방문) | 편의·익숙 | 약관상 **비상업용**(상업은 Pro $20/월) |
| **닷홈 등 한국형** | 일일 트래픽 제한 | 한국어·PHP 메일 | CDN 없어 느림·재사용 불리 |

- 성능이 셀링포인트(Lighthouse)면 **CDN 있는 쪽(Cloudflare)** 유리. 상업 약관 깔끔한 것도 Cloudflare.
- 어느 쪽이든 **고객 계정에 배포 = 호스팅비 0원·lock-in 0** 메시지는 동일.

### 문의 수집 (서버·DB 없음)
- **구글시트 자동 적재**(Google Apps Script 웹훅) — 문의가 시트에 쌓이고, **엑셀(xlsx)로 그대로 다운로드**. ★엑셀 요구 충족.
- 또는 **이메일 수신**(web3forms·Formspree 등 무료) — 폼 제출 시 메일로 도착.
- 별도 백엔드/DB 불필요 = 더 가볍고 저관리. (스팸은 허니팟으로 차단)

## 3. 상품 구성 (3-tier · 전부 100만 이하)

> 한 줄 콘셉트: **"30만원인데 300만원처럼 보이는, 스크롤로 압도하는 사이트. 고객님 무료 계정에 배포."**

> **⚠️ "섹션" 단위 정의 (혼선 방지)**: 모든 tier는 **총 섹션 수**로 작업량을 고정한다. 페이지로 나누든(메인+서브) 원페이지 풀스크롤로 만들든 **총 섹션 수는 동일**하다. 즉 "서브 4페이지 × 10섹션 = 40섹션" 같은 폭주는 없다. 페이지는 그저 섹션을 묶는 방식일 뿐, 견적 기준은 항상 총 섹션 수다.

### BASIC — 스크롤 원페이지 · **300,000원 · 3일 · 수정 1회**
- 1페이지 원스크롤 **총 4~6섹션**
- **오버스펙 애니 기본 세트**: 히어로 트랜지션 1 + 페이드업/시차 + 숫자 카운트업 + 핀 1개
- 반응형(PC·태블릿·모바일), **SEO/OG/파비콘**
- 전화·카카오 **고정 CTA 바**
- **무료 호스팅 배포(고객 계정)** + 기본 도메인
- 납품 후 **1개월 무상 하자보수**
- *문의폼 없음 — 보여주기·CTA(전화·카카오) 중심. 폼이 필요하면 옵션으로 추가*

### STANDARD — 스크롤 멀티 + 시그니처 애니 · **500,000원 · 5일 · 수정 1회** ⭐가장 인기
- **총 8~10섹션** (원페이지 풀스크롤 또는 메인+서브 분할 — 어느 쪽이든 총합 8~10섹션)
- **시그니처 애니 1종 선택**: 가로 스크롤 / 이미지 시퀀스 / SVG 선 그리기 / 카드 스태킹 중 택1 (오버스펙 포인트)
- 갤러리 + 후기 슬라이드 + 카카오맵
- **문의폼 → 구글시트 자동 적재(엑셀로 열람·다운로드) 또는 이메일 수신** + 스팸 차단
- 반응형·SEO·공유 최적화 전체
- **무료 호스팅 배포(고객 계정)**
- 납품 후 **1개월 무상 하자보수**

### PRO — 풀 인터랙션 · **800,000원 · 7일 · 수정 3회**
- **5페이지 이내 · 총 12~15섹션** (멀티페이지로 구성. 페이지를 몇 장으로 나누든 **작업량은 총 섹션 수(≤15)로 고정** — "5페이지 각 10섹션" 식 폭주 차단)
- **맞춤 시그니처 애니 2종**(3D 회전 / 이미지 시퀀스 / 패럴랙스 마스킹 / 스크롤 왜곡 등 레퍼런스 1:1)
- 문의 **구글시트 자동 적재 + 이메일 알림**(엑셀 다운로드 가능, 별도 DB·관리자 없음)
- 성능 최적화(Lighthouse 90+ 목표), 프리로더·전환 연출
- **무료 호스팅 배포(고객 계정)** + 운영 가이드 문서
- 납품 후 **1개월 무상 하자보수**

> tier 레버: **총 섹션 수 + 애니메이션 강도.** (illyweb 패턴 + 우리 강점) 회원/로그인/결제/복잡한 CMS는 **전 tier 미포함** = 관리 부담 차단.
