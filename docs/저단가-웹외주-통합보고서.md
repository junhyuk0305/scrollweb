# 저단가 웹 외주 — 통합 보고서

> **핵심 전략 문서.** 저단가(20~100만원, 평균 30~80만원) · **최소 기능** · **관리자 페이지 없음** · 폼까지만으로 납품 가능한 웹 외주 사업의 단일 기준 문서.
> 구성: ① 핵심 전략 → ② 저단가 사이트 30종(★) → ③ 차별화 → ④ 업종별 심화 리서치 → ⑤ 규제 → ⑥ 기능 모듈 → ⑦ 링크.
> (기존 [프로젝트개요.md](프로젝트개요.md)의 "프리미엄 스크롤트리거" 노선과 별개로, **최소 기능 저단가 노선**을 정의한다.)

# 0. 핵심 전략 (한 장 요약)

- **무엇을 파나**: 정적(static) 브로셔/원페이지 사이트. **관리자(CMS)·쇼핑몰·결제·DB 없음. 폼까지만.**
- **왜 이 구간**: "정적 = 관리자 불필요"라서 단가가 낮고, 내가 빠르게 만들 수 있다.
- **가격**: 30~80만원 고정가. (6~10만원 템플릿 수정은 마진 없음, 100만원+ 관리자/쇼핑몰은 안 받음)
- **납기**: 3~5일.
- **"관리자 없음"을 강점으로 판다**: 빠름 · 저렴 · 해킹/개인정보 사고 위험 0. 수정은 **월 유지보수 구독**으로 분리 = 반복 수익.

# 3. 공통 차별화 포인트 10 (저단가지만 경쟁 우위)

1. **고정가 패키지** — "30/50/80만원" 명확히. 견적 협상 피로 제거
2. **속도** — 3~5일 납품 보장
3. **모바일 100%** — 구식 업체의 1번 약점 정조준
4. **고정 CTA** — 전화·카톡·예약·견적 sticky (전환 직결)
5. **신뢰 신호 구조화** — 면허·인증·실적·후기 전면 배치
6. **업종 규제 안전버전** — 의료/법무/학원 광고규정 대응 = 경쟁사가 못 하는 차별화
7. **사진 정리/촬영 가이드** — 자영업은 좋은 사진이 곧 매출
8. **외부 연동(링크형)** — 네이버예약·캐치테이블·스마트스토어 "연결"만 (자체구현 X)
9. **SEO/AEO 기본** — 시맨틱 마크업, OG 태그, 네이버/카톡 공유 최적화
10. **유지보수 구독** — 콘텐츠 수정 대행 월 정액 = 반복 수익

쇼핑몰·결제 · 회원가입/로그인 · 자체 예약+결제 시스템 · 매물/상품 DB · 관리자 CMS 풀구현 · 실시간 재고 · 다국어 풀세팅 · 복잡한 API 연동(GitHub 잔디·실시간 화물조회 등)
