## 다시 만들기 (수정 후 재렌더)
```
cd docs/images
node shoot.mjs        # HTML 수정 후 실행 → out/ PNG 갱신
```
- 소스: `main-652x488.html`, `detail-01~09-*.html`, 공통 `image.css`
- `node_modules/`·`shoot.mjs`는 렌더용. 크몽엔 `out/`의 PNG만 업로드.

## 교체 포인트 (실사화 시)
- D1·D5·D6 디바이스 화면은 현재 추상 UI 프레임. 실제 데모 캡처/AI 이미지로 교체하려면
  해당 HTML의 `.ui-hero`·`.screen` 영역에 `<img>`를 넣고 재렌더.
- D5 데모 캡처 사용 시: picsum 스톡사진 교체, 가상 수치·별점 크롭 제외(기획서 §0 참고).
